- **永久留名**：被核心库采纳的 VSC 组合将以你的 GitHub ID 命名（如 `CV105_by_YourName`）。

1. **部署 SDK**：在你的设备上运行 AML SDK。
2. **开启遥测**（可选）：设置 `telemetry=True`，观察能效数据。
3. **发现高频组合**：在日志中找到出现频率最高的 VSC 序列。
